# Polygon Art Generator

## Overview
This Python program uses the Turtle Graphics library to generate colorful geometric art made of polygons.
Users can select from 9 different styles, and the program will automatically draw multiple random polygons on the screen.

---

## Features
- Draws triangles, squares, pentagons, or random polygons.
- Supports layered art (multiple nested polygons).
- Generates random colors, sizes, and positions for each shape.

---

## How to Run
1. Open a terminal in the project folder.
2. Run tests in your terminal by:
```bash
python polygon_art.py
```
